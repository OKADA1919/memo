# Raspberry Piのソフトウェアアップデート

## 1.現在のバージョン確認
```
uname -a
```

## 2.パッケージ情報を最新にする
```
sudo apt-get update
```

## 3.インストール済みのパッケージを最新にする
```
sudo apt-get upgrade
```

## 4.OSのディストリビューションを最新にする
```
sudo apt-get dist-upgrade
```

## 5.ファームウェアアップデート
```
sudo rpi-update
```

## 6.再起動
```
sudo reboot
```
